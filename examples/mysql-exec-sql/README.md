## Prerequisites

- Starting MySQL server is goddamn easy with:

```bash
docker compose up -d
```

- The `docker-compose.yaml` file and `.env` file are predefined in this repository; U can modify them to customize

## Try them on ur own

- Executing the script will demonstrate the exercise:

```bash
./01-transaction.sh
```
